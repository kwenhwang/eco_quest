# ECO_QUEST 프로젝트 문서

## 개요
이 문서는 ECO_QUEST 프로젝트의 기술 문서들을 포함하고 있습니다. 각 문서는 프로젝트의 특정 영역을 상세히 설명합니다.

## 문서 구조
- `api_spec.md`: API 엔드포인트 명세 및 사용 방법
- `db_design.md`: 데이터베이스 설계 원칙 및 구조
- `db_schema.md`: 데이터베이스 스키마 상세 정의
- `DBML_dbdiagram.io`: dbdiagram.io에서 사용되는 DBML 코드
- `development_process.md`: 개발 프로세스 및 워크플로우
- `game_design.md`: 게임 메커니즘 및 규칙 설계
- `how_to_run.md`: 프로젝트 실행 방법 가이드
- `how_to_test.md`: 테스트 실행 및 작성 가이드
- `technical_specifications.md`: 기술 스택 및 아키텍처 명세

## 문서 업데이트
- 모든 문서는 최신 상태를 유지해야 합니다
- 문서 수정 시 관련 팀원들과 리뷰를 진행해주세요
- 주요 변경사항은 커밋 메시지에 명확히 기록해주세요

## 문서 작성 가이드라인
- 명확하고 간결한 문장 사용
- 코드 예제는 적절한 포매팅과 주석 포함
- 필요한 경우 다이어그램이나 이미지 활용

3. 승인 후 main 브랜치에 병합
